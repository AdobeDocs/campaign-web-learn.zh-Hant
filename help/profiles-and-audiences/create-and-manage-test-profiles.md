# 建立和管理測試輪廓

在 Adobe Campaign 中建立行銷活動時，您可以在傳送的訊息內建立動態內容及個人化。測試輪廓可讓您仔細檢查您所傳送的訊息是否如預期般向使用者顯示。

了解如何在用戶端主控台中建立測試輪廓，以及如何在 Adobe Campaign Web UI 中管理和編輯輪廓。
