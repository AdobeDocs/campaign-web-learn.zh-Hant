# 建立直接郵件傳遞

直接郵件是一種離線頻道，可讓您產生檔案，以大量傳送個人化信件給您的客戶，例如明信片、傳單或目錄。 建立直接郵件傳送時，Adobe Campaign會自動產生擷取檔案，其中包含所有目標設定檔和選取的資料，例如郵寄地址和設定檔屬性。

了解如何在 Adobe Campaign v8 Web 使用者介面中建立直接郵件傳遞。
